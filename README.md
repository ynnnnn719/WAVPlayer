# WAV 音效播放器
這是一個WAV 音效播放器。  
使用者可以選擇 `.wav` 音效檔進行播放，並提供播放一次、重複播放、停止播放與結束程式等功能。

---

## 功能介紹

- 瀏覽並選擇 WAV 音效檔
- 播放一次音效
- 重複播放音效
- 停止播放
- 關閉程式確認提示

---

## 執行方式
1. 點擊「瀏覽」選擇 `.wav` 檔案
2. 使用按鈕控制播放功能

---

## 錯誤處理

若使用者未選擇正確的 WAV 檔案，程式會顯示提示訊息
